
# WIFI Password extractor

CLI tool for windows devices

written in GO lang
## Installation

### Clone project

```bash
  git clone https://github.com/Will-Bee/wifiPassExtractor.git
```

### Build

```bash
  cd buildScript
  build.bat
```

### Run that shi.

```bash
  main.exe
```

### Run that shi. but with colors

```bash
  main.exe -c
```
